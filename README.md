# Netflix_Content_Engagement_Viewer_Retention
Topic: Hooked or Dropped: Identifying Viewer Disengagement Risk & Content Opportunity Zones for Netflix. 

Goal: Discover patterns of viewer engagement, drop-offs, inefficiencies, and make powerful recommendations for Netflix.

💡 Objective:
To uncover trends in viewer engagement, identify drop-off risks, analyze underutilized content zones, and deliver actionable recommendations to Netflix using an engaging Power BI dashboard

📊 Dataset:
We’ll use this real Netflix dataset from Kaggle (same as your uploaded one):
🔗 [Netflix Movies and TV Shows – Kaggle Link] (https://www.kaggle.com/datasets/shivamb/netflix-shows)

Step 1: Define Dashboard Pages
We’ll create 4 report pages, each aligned with a unique view but designed in the same visual style:

Page Title	Purpose	Visual Theme
1. Global Overview	Engagement trend by release year, country, type	Total titles by year, genre heatmaps, filters
2. Drop-Off Risk Matrix	Identify content with high disengagement signals	Genre × Duration × Release Year matrix
3. Hidden Opportunity Zones	Uncover under-leveraged content with strong potential	Genre × Country views with KPIs
4. Recommendation Studio	Suggestions to reduce churn & improve stickiness	Strategic genre expansion, length targeting.

PAGE 2 TITLE:
